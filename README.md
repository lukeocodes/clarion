# Clarion

A macOS menu bar app that reads text aloud using [Deepgram's](https://deepgram.com) text-to-speech API. Select any text, right-click, and choose "Read Aloud" from Services. Supports 40+ voice models with automatic language detection.

## Install

```
make install
```

## Requirements

- macOS 14+
- Swift 6
- A [Deepgram API key](https://console.deepgram.com)
